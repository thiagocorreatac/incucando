# Teste Frontend Incuca

## O que foi feito
Eh uma SPA onde na primeira tela eh exibida uma informacao da aplicacao com um botao de acao que direciona para a proxima tela, na tela seguinte eh exibida a informacao que nivel de humor eh 100% triste e um botao para iniciar a melhoria do humor, na proxima tela, eh exibida mais uma ultima informacao entao eh iniciado uma sequencia de piadas, o usuario pode informar se gostou ou nao da piada, a conlusao acontece quando internamente a soma de avaliacao positivas nas piadas chega em 10, por fim, o usuario eh redirecionado para a tela de analise, com uma nova informacao e opcao para encerramento do teste.

## Como fiz
O projeto foi criado utilizando Vue.js e Vuetify, o gerenciamento de estados foi desenvolvido com vuex, axios para realizar as requisicoes, componentes bem segmentados e separacao de camadas para facilitar a implmentacao de novas funcionalidades, foi escolhido o Jest para a realizacao dos testes.
